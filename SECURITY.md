# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this package, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Email <security@spreedly.com> with details
3. Include steps to reproduce the issue
4. We will respond within 48 hours

For security architecture details, credential management, shared responsibility, and deployment guidance, see the [Security Guide](docs/security-guide.md).
